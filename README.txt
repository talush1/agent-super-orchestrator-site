
ASO Frontend – Step by step
===========================
1) Upload ALL files in this folder to your GitHub repo: agent-super-orchestrator-site (root).
2) Ensure GitHub Pages is ON (Settings → Pages → Build from main / root).
3) Open your site: https://talush1.github.io/agent-super-orchestrator-site/
4) Go to Settings page → enter your API Base URL (from backend deploy) → Save.
